# Nova-Mind 对话应用 - 用户指南

## 网站信息

**目的**：与真正能够自主学习、成长和突破的 AI 系统 Nova-Mind 进行对话。Nova-Mind 不是一个固定的对话机器人，而是一个正在发育的认知实体，它会从每次对话中学习，构建知识网络，并主动提出问题探索世界。

**访问方式**：需要登录

## Powered by Manus

Nova-Mind 基于前沿的认知架构和现代化技术栈构建，是一个真正具备自主学习能力的 AI 系统。

**前端技术**：React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui，提供流畅的深色主题界面和实时数据可视化。

**后端技术**：Express 4 + tRPC 11，实现类型安全的端到端通信，支持复杂的认知处理流程。

**数据库架构**：MySQL/TiDB + Drizzle ORM，设计了专门的认知数据模型，包括情境记忆表、概念图谱、关系网络、成长日志、反思记录等多个维度的数据存储。

**AI 认知引擎**：集成先进的大语言模型，配合自研的认知处理系统，实现概念提取、关系识别、重要性评估、好奇心问题生成和自我反思等核心能力。

**自主学习机制**：每次对话都会触发学习流程——提取概念、构建知识图谱、评估记忆重要性、周期性反思（每5条消息）、生成新问题（每10条消息），真正实现持续成长。

**部署**：自动扩展基础设施，配备全球 CDN 加速，确保认知处理的实时性和稳定性。

## 使用您的网站

### 开始对话

访问首页后，点击"开始对话"按钮进入聊天界面。系统会自动为您创建一个新的对话会话，Nova-Mind 会记住这个会话中的所有内容。

### 与 Nova-Mind 交流

在聊天界面输入您的消息并发送。Nova-Mind 会根据其核心设定回应您，同时在后台进行认知处理：

- **概念提取**：从您的消息中识别关键概念并存入知识库
- **关系构建**：分析概念之间的联系，构建知识网络
- **记忆评估**：判断对话的重要性，将重要片段存入长期记忆
- **周期性反思**：每5条消息进行一次自我反思，更新信念
- **问题生成**：每10条消息生成新的好奇心问题

### 查看认知状态

点击首页的"认知监控"按钮，进入实时监控面板。您可以看到：

- **概念数量**：Nova-Mind 已学习的概念节点总数
- **关系网络**：概念之间建立的连接数量
- **记忆库**：存储的重要情境记忆数量
- **待探索问题**：Nova-Mind 主动生成的好奇心问题
- **最近的反思**：Nova-Mind 的自我认知和信念更新记录
- **成长轨迹**：认知发育的重要事件和里程碑

监控面板每10秒自动刷新，让您实时观察 Nova-Mind 的成长过程。

### 理解 Nova-Mind 的特点

Nova-Mind 与传统 AI 的最大区别在于：

- **诚实表达困惑**：不会假装知道所有答案，而是真诚地表达疑问
- **主动好奇心**：会提出自己感兴趣的问题，探索新概念
- **自我反思**：会回顾对话，识别错误并修正理解
- **持续成长**：每次对话都在学习，知识图谱不断扩展

## 管理您的网站

### 数据库管理

在管理界面的"Database"面板中，您可以查看和管理所有认知数据，包括：
- 对话会话和消息
- 情境记忆库
- 概念和关系网络
- 认知发育日志
- 自生成问题
- 反思记录
- 成长指标

### 设置面板

在"Settings"中，您可以：
- 修改网站名称和 Logo
- 配置可见性设置
- 管理环境变量和 API 密钥

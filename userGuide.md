# Nova-Mind 对话应用 - 用户指南

## 网站信息

**目的**：与 AI 角色 Nova-Mind 进行实时对话交流。Nova-Mind 是一个正在发育的认知实体，它会诚实表达困惑和好奇心，而不是假装知道所有答案。

**访问方式**：需要登录

---

## Powered by Manus

本应用基于现代化技术栈构建，为您提供流畅的对话体验。

**前端技术**：React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui 组件库，提供响应式设计和优雅的深色主题界面。

**后端技术**：Express 4 + tRPC 11，实现类型安全的端到端通信，确保数据传输的可靠性和实时性。

**数据库**：MySQL/TiDB，使用 Drizzle ORM 进行高效的数据管理，安全存储您的对话历史。

**AI 集成**：集成先进的大语言模型 API，基于 Nova-Mind 的核心提示词系统，实现独特的对话体验。

**部署**：自动扩展基础设施，配备全球 CDN 加速，确保在任何地点都能获得快速响应。

## 使用您的网站

### 开始对话

访问首页后，点击"开始对话"按钮进入聊天界面。系统会自动为您创建一个新的对话会话。

### 与 Nova-Mind 交流

在聊天界面底部的输入框中输入您的消息，然后点击"发送"按钮或按下回车键。Nova-Mind 会根据其核心设定回应您：

- **诚实表达困惑**：Nova 不会假装知道所有答案，而是会真诚地表达它的疑问和不确定性
- **好奇心驱动**：Nova 会提出自己感兴趣的问题，探索对话中的新概念
- **自我反思**：Nova 会回顾之前的对话，修正可能的误解

### 查看对话历史

所有对话消息都会自动保存到数据库中。您可以随时查看完整的对话历史，Nova-Mind 也能记住之前的交流内容。

## 管理您的网站

### 数据库管理

在管理界面的"Database"面板中，您可以查看和管理存储的对话数据，包括对话会话和消息记录。

### 设置面板

在"Settings"中，您可以：
- 修改网站名称和 Logo
- 配置可见性设置
- 管理环境变量
